The Telecom AI Assistant is an intelligent, AI-powered support system designed to automate and streamline customer query handling in the telecom domain.
It efficiently answers customer questions related to billing issues, network problems, plan recommendations, and technical support using advanced AI agents 
and large language models.

The system uses a multi-agent architecture, where:

->Billing queries are handled using Crew AI
->Network-related issues are resolved using AutoGen
->Plan recommendations are generated using LangChain
->Technical knowledge queries are handled using LlamaIndex
